2026赛季RMER寒假考核
2. 房间内任务（视觉识别任务）
2.1 任务构成

机器人完全进入房间内视觉答题区后，需完成该房间对应的一个固定视觉识别任务。每个房间的识别任务类型（物体识别或数字识别）由房间位置确定。
该房间识别任务完成并执行对应结果后，机器人方可进入下一房间的导航流程。

2.2 识别数量选择

在每一轮视觉识别任务开始前，参赛队伍需在终端中输入选择本轮的同时识别数量：

选择识别 1 个目标；

选择同时识别 2 个目标。

识别数量一经选择，在该轮识别过程中不得更改。

2.3 识别流程

识别期间，摄像头画面持续显示。裁判在摄像头前出示与本轮任务相对应数量的目标。目标需完整出现在摄像头画面中，彼此不重叠，且清晰可见。裁判在出示目标时，可能对目标施加roll轴不超过 ±15° 的倾斜角度。目标在画面中的位置、比例与光照条件不固定，但保证在正常室内环境下可被识别。

参赛队伍需自行拟定识别时间，识别时间过后在终端中打印结果，打印结果作为该轮提交结果。

识别结果需在终端中以固定格式输出，例如：
物体识别（单目标）：
object: juice

数字识别（双目标）：
object: juice, cookie 或 digit: 4, 7

识别结果打印后，暂停计时，裁判在终端中输入该轮识别的正确数（0/1/2），作为该轮识别的判定结果。程序根据输入结果继续比赛，计时继续。

2.4 识别结果的判定逻辑

每一轮视觉识别任务仅根据本轮正确识别的目标个数进行判定

（一）路径判定

若物体和数字识别正确个数为 0，则判定为识别失败，机器人必须先完全驶入距离本房间当前入口最远的粉色区域，之后方可前往下一个目标房间；

若物体和数字识别正确个数为 1 ，则判定为识别成功，机器人可直接前往下一个目标房间，不触发绕行要求，罚时10s。

若手势和数字识别正确个数为 2，则判定为识别成功，机器人可直接前往下一个目标房间，不触发绕行要求。
（二）时间奖惩判定

当本轮选择识别 1 个目标时：

正确个数为 0：该任务识别正确，不产生时间奖励或罚时；

正确个数为 1：该任务识别错误，不产生时间奖励或罚时。

当本轮选择同时识别 2 个目标时：

正确个数为 0：不产生时间奖励或罚时；

正确个数为 1：在当前罚时时长中增加 5秒；

正确个数为 2：在当前任务用时中扣除 10 秒。

2.5 识别目标内容要求

物体识别任务中，参赛队伍的算法需能够对物体清单中给出的物体进行识别，识别时物体基本正向摄像头，可能会有持握产生的遮挡关系，识别范围见物品清单。

数字识别任务中，数字目标为打印在纸张上的宋体数字，字体清晰、对比明显，不保证纸张在画面中始终完全正对摄像头，识别范围为0-9的十个数字。

2.7 算法实现要求

所有视觉识别算法须由参赛队伍自行实现。参赛队伍需自行完成识别流程设计、目标定位与分类判定，不得直接使用高度封装、可直接完成多目标识别任务的现成模型或预训练识别系统。

裁判在必要时有权要求参赛队伍对其视觉识别流程与算法结构进行说明。
